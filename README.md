# Sistema de Gerenciamento de Pousada

Projeto desenvolvido como Trabalho Final da disciplina de Programação Orientada a Objetos (POO).

## Sobre o Projeto

O Sistema de Gerenciamento de Pousada foi desenvolvido com o objetivo de aplicar os conceitos de Programação Orientada a Objetos por meio da implementação de um sistema capaz de auxiliar na administração de uma pousada.

O sistema permite realizar operações relacionadas ao gerenciamento de hóspedes, quartos, reservas e demais informações necessárias para o funcionamento da pousada, oferecendo uma interface organizada e funcionalidades voltadas para a administração do estabelecimento.

## Funcionalidades

- Cadastro de hóspedes;
- Cadastro de quartos;
- Gerenciamento de reservas;
- Consulta, atualização e remoção de registros;
- Persistência de dados em arquivos;
- Leitura automática dos arquivos contendo os dados de hóspedes, acomodações e serviços ao iniciar o sistema;
- Atualização dos arquivos durante a execução da aplicação, garantindo que as alterações realizadas sejam armazenadas de forma permanente.

## Tecnologia Utilizada

- Java

## Conceitos de POO Aplicados

Durante o desenvolvimento foram utilizados diversos conceitos fundamentais da Programação Orientada a Objetos, como:

- Classes e Objetos;
- Encapsulamento;
- Herança;
- Polimorfismo;
- Abstração;
- Associação entre classes;
- Tratamento de exceções;

## Objetivo Acadêmico

Este projeto foi desenvolvido como avaliação final da disciplina de Programação Orientada a Objetos, tendo como finalidade consolidar os conhecimentos adquiridos durante a disciplina por meio da implementação de um sistema completo utilizando a linguagem Java.
